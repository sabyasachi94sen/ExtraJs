Position properties

  • Position Static: Position static is default properties of a element.Here Top,left,right and bottom properties wont work
  • Position Relative: It is the property of the element which allows to move the element wrt. its root position
  • Position Absolute: It is the property of the element which allows to move the element wrt. its immediate parent container
  • Position Fixed:It is the property which allows the element to remained fixed in that particular position when the page scrolls
  • Position Sticky:It is the property which combines position relative and sticky.The only thing is when it satisfy a CSS property its property starts working
